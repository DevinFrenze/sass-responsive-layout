Stretch and compress the screen width to see how the widths change. Content is dark gray. Containers are light gray.
Minimum Content Size (content--min): content always has gutter
content--min container--min
content--min container--padded
content--min container--max
Partial Content Size (content--partial): content uses a smaller gutter and until it expands to maximum width
content--partial container--min
content--partial container--padded*
content--partial container--max
Maximum Content Size (content--max): content is as wide as the maximum width
content--max container--min
content--max container--padded*
content--max container--max
Minimum Container Size (container--min): container is only as wide as its content
container--min content--min
container--min content--partial
container--min content--max
Padded Container Size (container--padded): container is as wide as maximum width and forces padding around content
container--padded content--min
container--padded content--partial*
container--padded content--max*
Maximum Container Size (container--max): container is as wide as screen
container--max content--min
container--max content--partial
container--max content--max
* "container--padded content--partial" is the same as "container--padded content--max"
